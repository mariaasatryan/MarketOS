# 🚨 СРОЧНОЕ ИСПРАВЛЕНИЕ: GitHub Pages 404

## Проблема:
Сайт показывает 404 "There isn't a GitHub Pages site here"

## 🔧 БЫСТРОЕ РЕШЕНИЕ:

### 1. Перейдите в настройки репозитория:
- Откройте: https://github.com/mariaasatryan/MarketOS
- Нажмите **"Settings"** (вкладка справа)

### 2. Настройте GitHub Pages:
- В левом меню найдите **"Pages"**
- В разделе **"Source"** выберите **"Deploy from a branch"**
- **Branch:** выберите **"main"**
- **Folder:** выберите **"/ (root)"**
- Нажмите **"Save"**

### 3. Альтернатива - GitHub Actions:
Если первый способ не работает:
- В разделе **"Pages"**
- **Source:** выберите **"GitHub Actions"**
- Нажмите **"Save"**

### 4. Проверьте Actions:
- Перейдите в **"Actions"** (вкладка в репозитории)
- Найдите **"Deploy to GitHub Pages"**
- Если не запускается: нажмите **"Run workflow"**

## ⏱️ Время активации:
- **2-5 минут** после настройки
- **Обновите страницу** через 5 минут

## 🎯 Ожидаемый результат:
После настройки сайт будет доступен по адресу:
**https://mariaasatryan.github.io/MarketOS/**

## 📞 Если ничего не помогает:
1. **Убедитесь, что репозиторий публичный**
2. **Проверьте права доступа**
3. **Попробуйте пересоздать репозиторий**
